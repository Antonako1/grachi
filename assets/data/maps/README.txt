When making a map, all of the images have to be the same size.
The image file has to be a bitmap (.bmp)

Types of map images
Map of country
Map with country's provinces
Map with country's provinces named
Map with country's rivers
Map with country's terrain

Recommended site for getting images from: https://d-maps.com/

colour values for terrain, from highest to lowest:
rgb(255,0,0)            RED     - Highest point
rgb(255,255,0)          YELLOW  - Second highest
rgb(255,0,255)          PINK    - Second lowest
rgb(0,0,255)            BLUE    - Lowest point
